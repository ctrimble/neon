Project Neon - Java Network Engineering

A library for supporting multiple URLStreamHandlers in the same JVM.

Download this project as a .zip file Download this project as a tar.gz file

Dealing with URL.setURLStreamHandlerFactory( URLStreamHandlerFactory handler )

This project addresses the problems caused by Java's URLStreamHandlerFactory singleton. Custom protocol handlers are a powerful mechanism for indirection in a Java program, but they are not usable in a shared environment. To overcome this limitation, this project translates java.net.URL constructor calls into static factory methods. These methods can then be isolated by a class loader.
